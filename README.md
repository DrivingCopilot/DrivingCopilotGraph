# On-Device Multimodal Driving Copilot

Welcome to the **On-Device Multimodal Driving Copilot** repository! This project implements an advanced, multimodal in-vehicle AI assistant leveraging LangChain and LangGraph to manage complex workflows, reasoning, and tool execution.

## Repository Structure

- **`graph/`**: Contains the core state graph definitions and RAG implementations.
  - `graph_rag.py`
  - `vector_rag.py`
  - *(Future)* Text2SQL agent implementation.
- **`services/`**: Holds service modules and data processing scripts, such as semantic chunkers, embedders, and PDF parsers.

## Upcoming Additions

**Knowledge Agent MCP Tools**: A dedicated folder will be created in the future to manage and house the comprehensive Model Context Protocol (MCP) tools for the Knowledge Agent.
